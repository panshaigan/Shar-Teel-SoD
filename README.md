# Shar-Teel SoD – Playable Shar-Teel NPC for Siege of Dragonspear

**Adds the iconic BG1 warrior Shar-Teel as a fully playable NPC companion during the Siege of Dragonspear campaign.**

Shar-Teel now joins your party in the events of *Siege of Dragonspear* (and in EET). She comes with her classic personality, new banters, romance options (if you have a compatible romance mod), custom scripts, and full voice-set integration.

## Compatibility
- **Baldur's Gate: Enhanced Edition** + **Siege of Dragonspear** DLC
- **Enhanced Edition Trilogy (EET)** (highly recommended)
- Works with Project Infinity and most other WeiDU mods (recommended install order: after core EET components, before heavy NPC overhauls)
- WeiDU v246+ required

## Installation
1. Download the latest release from the **Releases** tab (or the `.iemod` file for Project Infinity).
2. Extract the `SharTeelSoD` folder into your game directory.
3. Run `setup-SharTeelSoD.exe` (Windows) / `.sh` (Linux) / `.command` (macOS) and follow the prompts.
4. **Full documentation** is inside the mod: `SharTeelSoD/readme-SharTeelSoD.html`

## Components
- **Install Shar-Teel as playable NPC** (main component – includes dialogues, scripts, portraits, and SoD-specific content)
- Optional: Extra banters / voice-set tweaks (if you want them)
